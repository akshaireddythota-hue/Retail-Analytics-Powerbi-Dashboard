
# 📊 Retail Analytics & Performance Dashboard

## 🧾 Project Overview
This project presents an interactive retail analytics dashboard built using Power BI. It provides a comprehensive view of business performance by analyzing sales, customer behavior, product categories, and regional trends.

The dashboard is designed to help stakeholders make informed, data-driven decisions through clear visual storytelling and key performance indicators.

---

## 🎯 Objectives
- Monitor overall sales performance
- Identify top-performing products and categories
- Analyze customer segments and behavior
- Evaluate regional and store-level performance
- Track key KPIs such as return rate and average order value

---

## 📌 Key Features
- 📈 Sales trend analysis by month
- 🛍️ Product category performance breakdown
- 💳 Payment method distribution
- 👥 Customer segmentation (age, gender, loyalty)
- 🌍 Store location performance insights
- 📊 Sales representative comparison
- 🔄 Quarter-over-quarter performance analysis

---

## 📊 Dashboard Structure

### 🔹 Page 1: Sales Overview
- Total Sales, Orders, Return Rate, Avg Order Value
- Monthly sales trends
- Sales by product category
- Payment method analysis

### 🔹 Page 2: Product Insights
- Q1 vs Q2 comparisons (Sales, Orders, AOV, Return Rate)
- Product-level performance table
- Quantity vs Sales correlation

### 🔹 Page 3: Customer & Regional Insights
- Sales by age group
- Orders by gender
- Loyal customer distribution
- Sales and returns by location

## 📊 Key Insights
- The electronics category contributes the highest revenue
- Sales show a peak during Q2
- Credit cards are the most preferred payment method
- Dublin generates the highest sales among all locations
- Customer loyalty is strongest in the Electronics category
